Localização UFT/Campus Palmas/Bloco 3, Sala 12. Área 100m² Capacidade 44 pessoas
Descrição Laboratório de uso exclusivo do Curso de Ciência da Computação que atende às disciplinas dos períodos iniciais (1, 2 e 3), as quais possuem uma média de 40 acadêmicos matriculados. Ele é usado para práticas de programação e estrututuras de dados. Este laboratório fica acessível aos acadêmicos nos períodos matutino, vespertino e noturno, sendo um ponto de referência para os conteúdos práticos para os acadêmicos das turmas iniciais.
